# Weather Diary - Дневник погоды 🌤️

**Автор:** Масленников Захар

## 📝 Описание программы

Weather Diary — это графическое приложение на Python для ведения дневника погоды. Программа позволяет:

- ✅ Добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков
- ✅ Просматривать все записи в удобной таблице
- ✅ Фильтровать записи по дате и по температуре (выше заданного значения)
- ✅ Удалять выбранные записи
- ✅ Сохранять данные в JSON-файл и загружать из него
- ✅ Проверять корректность ввода (формат даты, тип температуры, непустое описание)

## 🚀 Как запустить приложение

### Требования:
- Python 3.7 или выше
- Стандартные библиотеки (tkinter, json, datetime, os)

### Инструкция по запуску:

1. **Скачайте файл** `weather_diary.py`

2. **Запустите приложение** любым способом:
   - **Windows:** Дважды кликните по файлу `weather_diary.py`
   - **macOS:** Дважды кликните или нажмите `Cmd + O` в Python Launcher
   - **Linux:** Правой кнопкой → "Открыть с помощью" → "Python"

3. **Используйте программу** через графический интерфейс

## 📖 Примеры использования

### Тест 1: Добавление новой записи
1. Введите дату: `2025-05-09`
2. Введите температуру: `18.5`
3. Введите описание: `Солнечно, легкий ветер`
4. Отметьте "Осадки", если нужно
5. Нажмите **"Добавить запись"**

### Тест 2: Фильтрация записей
1. В поле "Фильтр по дате" введите `2024-06-10`
2. Нажмите **"Применить фильтр"** — покажутся только записи за этот день
3. Для сброса нажмите **"Сбросить фильтр"**

### Тест 3: Фильтр по температуре
1. В поле "Температура выше" введите `10`
2. Нажмите **"Применить фильтр"** — покажутся записи с температурой > 10°C

### Тест 4: Проверка валидации
1. Попробуйте добавить запись с:
   - Неверной датой: `32.13.2024`
   - Текст вместо температуры: `тепло`
   - Пустым описанием: ` `
2. Программа покажет сообщение об ошибке

### Тест 5: Сохранение и загрузка
1. Добавьте несколько записей
2. Нажмите **"Сохранить в JSON"**
3. Закройте и снова откройте программу
4. Данные автоматически загрузятся из файла

### Тест 6: Удаление записей
1. Выберите любую запись в таблице (кликните по ней)
2. Нажмите **"Удалить выбранную запись"**
3. Подтвердите удаление
